
# Webpage for CYF – The Joy of Coffee

This project is a simple webpage created to showcase the joy and culture of coffee.  
It was built as part of a learning exercise in HTML and CSS, focusing on clean structure, styling, and presentation.

---

## 🌟 Features
- **HTML Content**: Includes headings, paragraphs, and lists to structure information about coffee.
- **CSS Styling**: Custom styles for font, margins, line height, background color, and text color.
- **Coffee List**: An unordered list of popular coffee drinks.
- **Images Section**: Space reserved for showcasing coffee-related images.
- **Responsive Design**: Easy to extend with responsive CSS for mobile-friendly layouts.
